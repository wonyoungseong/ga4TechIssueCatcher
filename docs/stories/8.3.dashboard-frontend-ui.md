# Story 8.3: Dashboard Frontend UI

## Status

Completed

## Story

**As a** 디지털 애널리틱스 팀원,
**I want** 웹 대시보드에서 검증 결과를 확인하기를 원합니다,
**so that** 이슈를 빠르게 파악하고 대응할 수 있습니다.

## Acceptance Criteria

1. ✅ 결과 목록 테이블이 표시된다 (속성명, 상태, 이슈 개수, 시간)
2. ✅ 날짜별 필터링이 가능하다
3. ✅ 상태별 필터링이 가능하다 (전체, 성공, 실패)
4. ✅ 검색 기능이 구현된다 (속성명, 계정명)
5. ✅ 상세 뷰에서 이슈 내용을 확인할 수 있다
6. ✅ 스크린샷을 클릭하여 확대 조회할 수 있다
7. ✅ 실시간 업데이트가 반영된다
